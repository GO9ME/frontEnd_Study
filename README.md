## 변수 사용법

- state : js 변수는 UI 갱신하지 못함, 값이 바뀌면 UI갱신
- props : 부모컴포넌트에서 자식컴포넌트 넘겨지는 데이터 js destructur
- context : 최상위 컴포넌트 지정한 값 ⇒ 어느 하위 컴포넌트에서 사용 가능
